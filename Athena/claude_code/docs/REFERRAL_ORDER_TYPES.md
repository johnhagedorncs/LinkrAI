## Usage Notes

### How to Search for Referral Types

```python
from athena_api import AthenaWorkflow

workflow = AthenaWorkflow(practice_id="195900")
referral_types = workflow.get_referral_order_types("cardio")

for ref_type in referral_types:
    print(f"{ref_type['ordertypeid']}: {ref_type['name']}")
```

### How Referral Workflow Uses This

The `execute_complete_referral_workflow()` function automatically searches for the appropriate referral type based on the specialty you provide:

```python
# User provides specialty
specialty = "cardiology"

# Workflow searches using SPECIALTY_MAPPINGS
search_term = SPECIALTY_MAPPINGS.get(specialty, {}).get("searchterm", specialty)

# Gets matching referral types
referral_types = workflow.get_referral_order_types(search_term)

# Uses first match
order_type_id = referral_types[0]['ordertypeid']
```
